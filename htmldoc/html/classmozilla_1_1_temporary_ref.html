Detailed Description

template<typename T>
class mozilla::TemporaryRef< T >

TemporaryRef<T> represents an object that holds a temporary reference to a T. TemporaryRef objects can't be manually ref'd or unref'd (being temporaries, not lvalues), so can only relinquish references to other objects, or unref on destruction.
