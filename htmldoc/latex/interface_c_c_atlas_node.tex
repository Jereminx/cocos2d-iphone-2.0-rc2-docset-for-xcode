\subsection{Detailed Description}
\hyperlink{interface_c_c_atlas_node}{C\-C\-Atlas\-Node} is a subclass of \hyperlink{class_c_c_node}{C\-C\-Node} that implements the \hyperlink{protocol_c_c_r_g_b_a_protocol-p}{C\-C\-R\-G\-B\-A\-Protocol} and \hyperlink{protocol_c_c_texture_protocol-p}{C\-C\-Texture\-Protocol} protocol

It knows how to render a Texture\-Atlas object. If you are going to render a Texture\-Atlas consider sub-\/classing \hyperlink{interface_c_c_atlas_node}{C\-C\-Atlas\-Node} (or a subclass of \hyperlink{interface_c_c_atlas_node}{C\-C\-Atlas\-Node})

All features from \hyperlink{class_c_c_node}{C\-C\-Node} are valid, plus the following features\-:
\begin{DoxyItemize}
\item opacity and R\-G\-B colors 
\end{DoxyItemize}
