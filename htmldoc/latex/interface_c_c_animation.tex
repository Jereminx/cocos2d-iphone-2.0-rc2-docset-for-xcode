\subsection{Detailed Description}
A \hyperlink{interface_c_c_animation}{C\-C\-Animation} object is used to perform animations on the \hyperlink{class_c_c_sprite}{C\-C\-Sprite} objects.

The \hyperlink{interface_c_c_animation}{C\-C\-Animation} object contains \hyperlink{interface_c_c_animation_frame}{C\-C\-Animation\-Frame} objects, and a possible delay between the frames. You can animate a \hyperlink{interface_c_c_animation}{C\-C\-Animation} object by using the \hyperlink{interface_c_c_animate}{C\-C\-Animate} action. Example\-:

\mbox{[}sprite run\-Action\-:\mbox{[}\hyperlink{interface_c_c_animate}{C\-C\-Animate} action\-With\-Animation\-:animation\mbox{]}\mbox{]};
