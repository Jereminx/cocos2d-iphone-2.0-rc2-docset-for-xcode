\subsection{Detailed Description}
Executes an action in reverse order, from time=duration to time=0

\begin{DoxyWarning}{Warning}
Use this action carefully. This action is not sequence-\/able. Use it as the default \char`\"{}reversed\char`\"{} method of your own actions, but using it outside the \char`\"{}reversed\char`\"{} scope is not recommended. 
\end{DoxyWarning}
