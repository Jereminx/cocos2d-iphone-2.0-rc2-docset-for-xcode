\subsection{Detailed Description}
\hyperlink{interface_c_c_transition_page_turn}{C\-C\-Transition\-Page\-Turn} transition. A transition which peels back the bottom right hand corner of a scene to transition to the scene beneath it simulating a page turn

This uses a 3\-D\-Action so it is strongly recommended that depth buffering is turned on in \hyperlink{class_c_c_director}{C\-C\-Director} using\-: \begin{DoxyVerb}   [[CCDirector sharedDirector] setDepthBufferFormat:kCCDepthBuffer16];\end{DoxyVerb}
