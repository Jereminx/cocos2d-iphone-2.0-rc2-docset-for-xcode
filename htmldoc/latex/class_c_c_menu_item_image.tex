\subsection{Detailed Description}
\hyperlink{class_c_c_menu_item_image}{C\-C\-Menu\-Item\-Image} accepts images as items. The images has 3 different states\-:
\begin{DoxyItemize}
\item unselected image
\item selected image
\item disabled image
\end{DoxyItemize}

For best results try that all images are of the same size
